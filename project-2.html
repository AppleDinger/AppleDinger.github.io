Kartavya Command System

A full-stack situational awareness platform utilizing FastAPI and PostgreSQL to manage police unit tracking, dynamic geofencing, and secure communication.

Live Demo

Project Overview

Kartavya is a specialized Command & Control system designed for the Goa Police Department. It replaces legacy manual tracking with a real-time digital dashboard, allowing supervisors to monitor field units across North and South Goa jurisdictions.

The core engine features a dynamic geofencing algorithm that automatically detects when officers leave their assigned zones. It also includes a Broadcast Ping System for rapid communication during high-priority alerts.

Key Engineering Features

  • FastAPI Backend: A high-performance REST API handling asynchronous location updates and spatial calculations.
  • Secure Authentication: Implemented Argon2 hashing to replace standard Bcrypt, ensuring industry-grade password security on cloud infrastructure.
  • Interactive Dashboard: A Streamlit frontend integrated with Folium maps for visualizing live unit positions and historical paths.
  • Cloud Architecture: Fully deployed on Render, utilizing a managed PostgreSQL database for persistent storage of logs and personnel records.
