
# Ram Mishra's Portfolio
Welcome to the repository for my personal portfolio website. This site serves as a central hub to showcase my technical projects, software architecture research, and skills as a Computer Science student at NIT Goa.

Live Site: https://appledinger.github.io/

## Tech Stack of the potfolio
* Frontend: HTML5, CSS3 , and Vanilla JavaScript.

* Interactivity: Custom JavaScript for mobile navigation and project image carousels.

* Contact System: Integrated with Formspree for seamless email delivery of user messages.

* Deployment: Hosted on GitHub Pages.

## Featured Projects
### 1. Smart Quiz Master
An AI-driven educational platform that generates interactive quizzes from PDFs and YouTube videos.

* Technologies: React (Vite), Node.js, MongoDB, and Gemini LLM.

* Key Feature: Leverages Large Language Models to extract key concepts and automate assessment generation.

### 2. Kartavya Command System
A high-performance public safety platform engineered for real-time police unit tracking and geofencing.

* Technologies: FastAPI, PostgreSQL (on Render), Streamlit, and Folium.

* Security: Implemented Argon2 hashing for industry-grade password protection.
