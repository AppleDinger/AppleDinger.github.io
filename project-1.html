Smart Quiz Master

A full-stack AI application that generates interactive quizzes from PDFs and YouTube videos, built with React, Node.js, and Gemini.

Live Demo

Project Overview

Smart Quiz Master is a study aid that allows users to generate quizzes instantly from their own learning materials. Whether it is a lecture PDF or an educational YouTube video, the app extracts the key concepts and uses AI to challenge the user's understanding.

Technical Architecture

The application is built on a robust architecture:

  • Frontend: Developed using React and Vite for a fast, responsive user interface.
  • Backend: A Node.js server hosted on Render handles API requests and data processing.
  • AI Engine: Integrated directly with the Gemini LLM to process text and generate intelligent questions.
  • Database: MongoDB is used to store user data and quiz history.
